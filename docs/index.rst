Bienvenue dans la documentation du **Projet 2** ! 🚀

Ce projet implémente une architecture microservices complète avec :

* **Backend** : API FastAPI pour la gestion de données
* **Frontend** : Interface Streamlit interactive
* **Base de données** : PostgreSQL avec persistance
* **Orchestration** : Docker Compose
* **CI/CD** : GitHub Actions avec tests, sécurité et déploiement
