=======
Secrets
=======

Configuration des secrets GitHub pour le CD.

Configuration DockerHub
=======================

1. Créer un token d'accès sur DockerHub
2. Aller dans GitHub Settings > Secrets
3. Ajouter les secrets :

   * ``DOCKERHUB_USERNAME`` : Votre nom d'utilisateur
   * ``DOCKERHUB_TOKEN`` : Le token d'accès

Documentation Complète
======================

Voir le fichier ``GITHUB_SECRETS.md`` à la racine du projet pour des instructions détaillées.
